** Breakout! **

Play the classic Atari game breakout using this Python app!

** How to Start **
Make sure all files are located in the same directory.
Run the main.py from an IDE or the terminal to start the program.

Package requirements:
 - Turtle
 - Time

** How to Play **
Use the left and right arrow keys to move your paddle across the screen.
You want to use your paddle and hit the ball back into the blocks to increase your score.
You'll have 3 lives to try and clear the board of all the blocks.
Clear all the blocks to win the game!
